Dinero v8.0.0-rc17 — Dynamic P2P release candidate

Run Dinero. Help the network.

Dinero is moving from a small fleet of bootstrap servers toward a self-healing peer-to-peer network. rc17 ships the current desktop wallets, relay/NAT traversal work, Dynamic P2P visibility, and the node dashboard pieces that let users see how their node participates.

What changed in rc17

This release candidate is about making Dinero feel less like four main servers and more like a peer network normal users can join.

Relay / NAT traversal

Nodes behind routers that reject direct inbound port mapping can still participate through relay-backed inbound paths. Fleet relays are bootstrap helpers, not the end-state.

  • Relay-capable peers advertise NODE_RELAY.
  • Wallet/mining mode can signal relay participation to the embedded daemon.
  • Relay hints refresh and expire so stale paths heal instead of lingering forever.

Dynamic P2P visibility

The daemon now exposes a peer-governor view so users and operators can see how the node discovers, ranks, and rotates peers over time.

  • getnetworkinfo reports Dynamic P2P mode and relay candidates.
  • Fleet nodes run active_slow_churn with relay enabled.
  • The Qt node dashboard consumes the new daemon RPCs.

Run a node

Datadirs follow the native convention for each install mode. The important part is simple: run the wallet or daemon, keep it online, and let it participate in peer discovery.

Platform / mode Default datadir Use case
Windows desktop / Core zip %APPDATA%\Dinero Windows application-data convention
macOS desktop ~/Library/Application Support/Dinero macOS application-data convention
Windows Server installer C:\ProgramData\Dinero Windows Service convention; survives uninstall
Linux packaged service /var/lib/dinero FHS/systemd daemon under the dinero service user
Linux source build / developer ~/.dinero Standard Linux CLI/daemon dot-directory convention
Linux GUI (AppImage / Qt tarball / dinero-qt-desktop .deb) ~/.dinero GUI auto-starts the bundled daemon in the user's home

Ubuntu node quick start

curl -fsSL https://dinerolabs.org/install.sh | sudo bash
sudo systemctl status dinero --no-pager
dinero-cli getnetworkinfo

macOS daemon quick start

tar -xzf dinerod-8.0.0-rc17-macos-arm64.tar.gz
cd dinero-operator-v8.0.0-rc17-macOS-arm64
./bin/dinerod
./bin/dinero-cli getnetworkinfo

What is still release-candidate

  • Windows v8 installers are not yet Authenticode-signed — SmartScreen may warn. Verify the SHA256 above and click "More info" → "Run anyway".
  • Linux rc17 packages are preview artifacts for Ubuntu 24.04+ x86_64 and glibc-compatible desktop systems.
  • Dynamic P2P is active in conservative slow-churn mode. It is designed to improve peer diversity gradually, not churn aggressively.
  • Relay/NAT traversal is useful infrastructure, but the long-term goal is broader community-run peers and relays, not dependence on Dinero Labs bootstrap nodes.
  • Stable promotion waits for the next final-release ceremony.
